# Python Data Cleaning Project
This project using Python in Jupyter Notebooks showcases my data cleaning abilities in Python using Pandas.
The dataset I have used for this project can be found [here](https://www.kaggle.com/datasets/desolution01/messy-employee-dataset).

My code fixes issues such as compound columns that can be split and inconsistent formatting. I have also reformatted the phone numbers as an international format, which makes them easier to read. The idea behind this project is to process the data so that it can be used further for multiple purposes: e.g. exploratory data analysis, visualisations, grouping and sorting data.
